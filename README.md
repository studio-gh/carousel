🏃‍♂️ Ritmo & Fôlego (v5.0) — Rumo à Meia Maratona do Rio 2027

Este é o repositório do Ritmo & Fôlego, um aplicativo web interativo de alta performance focado em corrida de rua de fundo e treinamento de força concomitante, projetado sob medida para um atleta de 41 anos, pai ativo e praticante diário de Yoga.

O objetivo final do planejamento tático contido neste software é a conclusão e excelente performance na Meia Maratona do Rio de Janeiro em Maio de 2027.

🎯 Funcionalidades Principais

📅 Planejamento de Corrida em 4 Fases: Periodização tática progressiva de 10 meses estruturada desde a base cardiovascular até o polimento final (tapering).

🧘‍♂️ Ativação Diária Integrada (Yoga): Rotina diária obrigatória de 20-25 minutos de Yoga e alongamento ativo pré-atividade, reduzindo o risco de lesões articulares e canelite.

💪 Módulo de Força Concomitante Adaptável (Dad-Friendly): Chaveador em tempo real para alternar entre rotinas de Kettlebell e Calistenia (Peso Corporal) — ideal para treinos rápidos e silenciosos em casa enquanto o bebê dorme.

✨ Treinador IA Gemini Integrado: Um coach virtual de alta precisão treinado para analisar RPE (esforço percebido), noites mal dormidas e feedback biomecânico de passada.

📊 Análise de Histórico Strava: Upload direto de arquivo .csv extraído do Strava com calibrador automático de ritmo de rodagem Z2 baseado nos últimos 6 meses.

🔬 Fundamentação Científica

A planilha e os algoritmos de zona de ritmo do aplicativo utilizam as melhores práticas de medicina do esporte e fisiologia de esforço:

Fartlek Sueco (Andres, 2024): Uso do "jogo de velocidades" para treinar a transição do sistema aeróbico e anaeróbico e melhorar a economia de corrida sem fadiga neurológica crônica.

Desenvolvimento Aeróbico de Longo Prazo (Guilherme, 2004): Ênfase no acúmulo gradual de volume (Zona 2) para promoção de capilarização periférica e biogênese mitocondrial.

Força Concomitante (Balsalobre-Fernández, 2016): Treinamento resistido focado em cadeia posterior e padrões unilaterais para aumento da rigidez miotendínea e economia de esforço em atletas de faixa etária 40+.

💻 Como Rodar o Projeto Localmente

Se você deseja executar o projeto em sua máquina local para testes ou desenvolvimento:

Clone o repositório:

git clone [https://github.com/seu-usuario/ritmo-folego.git](https://github.com/seu-usuario/ritmo-folego.git)


Instale as dependências:

npm install


Inicie o servidor de desenvolvimento:

npm run dev


🚀 Publicação & Deploy

Este projeto é totalmente compatível para compilação na nuvem e deploy estático gratuito:

Vercel: Conecte o repositório diretamente no painel da Vercel para obter um link seguro de produção atualizado a cada alteração de código.

GitHub Pages: Configure o homepage no package.json e publique usando npm run deploy.
